## Full Description (Both Stores)

Paste the same description for both:

```
X-Split turns long X/Twitter posts into clean, numbered threads — automatically.

HOW IT WORKS
- Install the extension
- Go to x.com and open the compose box
- Type or paste your text past 280 characters
- A sidebar appears showing each chunk with a copy button
- Copy and paste each chunk into X's compose box

That's it. No buttons to click. No settings to configure. No accounts to create.

FEATURES

Smart splitting — Breaks at sentence boundaries (. ! ?) when possible, falls back to word breaks. Your threads read naturally, not chopped mid-word.

Auto-split on input — The sidebar appears as soon as your text exceeds 280 characters. Edit or delete text and it stays in sync.

URL-aware counting — URLs are counted as 23 characters (matching X's t.co behavior). The extension never splits a URL across two chunks.

Per-chunk numbering — Each chunk gets (1/5), (2/5) etc. Choose from parenthesis, brackets, or slash format in settings.

Fill mode — Optional toggle to maximize character usage per chunk. Off by default (clean sentence breaks preferred).

Re-split — Edit your text after splitting, click the button to re-split. Don't start over.

Dark mode — Follows your system preference automatically.

Collapsible sidebar — Click X to hide the sidebar. A floating tab stays on screen to reopen it when you need it.

Customizable — Click the extension icon in your toolbar to adjust max characters, numbering style, URL counting, and fill mode.

WHY X-SPLIT
- Free — No subscription, no API fees, no hidden costs
- Private — All processing happens in your browser. Zero data collection. No analytics.
- Lightweight — Vanilla JavaScript, zero dependencies, no build step
- Works inside x.com — No tab switching, no external editors

PRIVACY
X-Split does not collect, store, or transmit any personal data. All text processing is local. No third-party services. No analytics. No tracking.

Full privacy policy: https://mujtabaj30.github.io/x-split/privacy/
```
