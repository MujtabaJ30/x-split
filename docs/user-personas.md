# User Personas

## Persona 1: Sarah Chen -- The Writer

**Tagline:** "I write long threads. I don't want to pay X to do it."

**Bio:** 34, freelance tech journalist. Writes 3-5 analysis threads per week on AI, startups, and developer tools. Has 12k followers. Refuses to pay for X Premium on principle. Currently splits threads manually in a text editor.

**Goals:**
- Post long-form analysis as clean multi-post threads
- Maintain professional readability -- no mid-sentence cuts
- Spend less time on mechanics, more on writing

**Pain Points:**
- Manual splitting takes 10-15 mins per thread
- X's compose box gives no help past 280 chars -- just stops typing
- Has accidentally posted incomplete posts because she miscounted
- Copy-pasting between Notes app and X is friction she hates

**How X-Split Helps:**
- One click splits her draft into clean sentence-bounded chunks
- Numbering is automatic -- no manual `(cont.)` tracking
- Preview lets her verify flow before posting
- No new tool to learn -- it lives inside X's compose box

## Persona 2: Marcus Johnson -- The Indie Hacker

**Tagline:** "Building in public means writing threads every day."

**Bio:** 28, solo founder of a devtools SaaS. Posts daily "building in public" threads detailing feature work, metrics, and lessons. Writes on mobile and desktop. 5k followers. Needs threads to look polished because they double as marketing.

**Goals:**
- Post daily threads without it feeling like a chore
- Keep thread formatting consistent (same numbering style, clean breaks)
- Focus on content, not mechanics

**Pain Points:**
- X's 280-char limit breaks his flow -- he writes then has to restructure
- Thread numbering is error-prone. Has posted `(2/3)` then `(4/4)` before.
- Tried Chirr App but hates switching tabs
- Typefully is too expensive ($18/mo for features he doesn't need)

**How X-Split Helps:**
- Split happens in the compose box -- no tab switching
- Consistent `(1/N)` numbering every time
- Copy button for each chunk means less friction per post
- It's free. No subscription. No trial.
